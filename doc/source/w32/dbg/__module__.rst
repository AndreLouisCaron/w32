.. module:: w32::dbg
   :synopsis: Debugging and diagnostics
.. moduleauthor:: Andre Caron <andre.l.caron@gmail.com>

This module provides services for debugging applications and performing
diagnostic in situations where debugging is not practical or even possible.

.. note::

   Although this module aims at providing debugging services, it needs not be
   only used in debugging and testing. Actually, shipping applications with
   *some* diagnostic capabilities is *highly* recommended.
